The project is done using python
The code editor tool used for this project was jupyter notebook
The core idea of the project is to understand the basics of machine learning.
Uses a dataset to detect faults using 3 various machine learning models and compares them in terms of accuracy , precision , recall and f1 score
Displays plots that compare the training and testing of the models on datasets. 
A confusion matrix of the most accurate model random forest is also produced
Uses libraries such as pandas, numpy , matplot library etc for loading the data, performing mathematical operations and plotting the visualisations.
